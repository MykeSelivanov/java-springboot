# jd-java-springboot
Cheat-sheet repository for building SpringBoot applications <br>
Module names are self explanatory

## Technologies
- Spring
- SpringBoot
- Spring-MVC
    - server side rendering with Thymeleaf
- Spring-ORM
    - Derived queries
    - JPQL
    - Native queries
    - Named queiries
    - SQL Triggers
- Spring-Security
    - Default Config / Custom Config
    - Form Authorization
    - HttpBasic
    - JWT
- Spring-Rest
    - Jackson
    - Response Entity
    - Rest Template
    - Rest Security with JWT
    - OpenAPI 3.0 (Swagger)
- Aspected Oriented Programming with Spring
- Unit and Integration testing
    - JUnit
    - Mockito
    - MockMVC
